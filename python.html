2 Printing to the screen

  1. The print() function displays a message on the screen. print(string1, string2, ...)
    1. Try adding the line print("Hello World!") within the test() function, save the file with Ctrl+S and click the refresh button in the preview window.
    2. print() tells Python what it is supposed to do (in this case, print something out to the screen).
    3. "Hello World!" is a string, a text phrase enclosed in quotes. print() can output string values, as well as variables, as you'll see in the next section.
    4. You can combine multiple string values into a single string with the + sign. Try print("super"+"cali"+"fragi"+"listic"+"expi"+"ali"+"docious")!
    5. When passing in multiple values to print(), they are separated by spaces automatically. Try print("I'm", "a", "really", "long", "sentence!")

3 Variables

  1. Variables are boxes in the computer’s memory that store single values. You give each variable a name and use that name to refer to the value stored in that box. Variable names can only contain letters, numbers and the underscore _.
  2. To create a variable, give the variable a name and assign it a value using the = sign.
    1. Storing strings: my_name = "Thanos" means that we have made a variable called my_name and given it a text value of Thanos.
    2. Variables can store numbers and expressions that evaluate to numbers too.
      For example, body_temp = (98.6 - 32) * 5/9 evaluates the expression to 37.0, and stores it in body_temp.
    3. One more useful value type we can store is a boolean, either True or False.
      Check the section on comparisons for some expressions that can evaluate to boolean values.

4 If statements

  1. In Python you can use if statements to run commands under a certain condition. For example:
    age = 14
    if age > 20:
        print("You are old!")
    Pro Tip! You can use <, > or == to make comparisons. Notice that in an if statement, you have to use double "equals" sign. One "equals" sign tells Python to assign a value to something (x = 5 assigns the value of 5 to the variable x). Two "equals" signs means that you are checking two things to see if they are equal (age == 20 checks to see if the value of the age variable is 20).
  1. If-else statements are an extended form of if statements. The else statement tells Python what should happen if the if condition is not true.
  2. class_size = 20
    apples = 0
    if apples >= class_size :
        print("I have enough apples for everybody!")
    else:
        print("I don't have enough apples!")
Pro Tip! You are not limited to just checking to see if a variable is equal to a certain value. You can check for many different things. See the Comparison section for more details.

5 For Loops

  1. for loops are used when you want instructions to repeat a certain number of times.
    For example, if we wanted to print "Hello World!" 5 times:
    1. for i in range(5):
          print("Hello World!")
    2. range(5) gives a sequence of numbers starting from 0 and up to (but not including) 5. Each iteration of the loop, it saves the number into variable i. Try replacing "Hello World!" by i and see what happens!

6 Bonus: Turtles!

  1. Our code uses turtle graphics, a python library, to draw fireworks onto the Thimble webpage. Turtles are digital animals in Python that can move anywhere on the screen.
    Within the function draw_house(house_turtle), we've given you a turtle for you to draw a house with - it's the black arrow in the center of the screen. Here are some simple commands that you can call on the turtle:
    1. house_turtle.forward(distance) makes the turtle walk forward by distance, a number you provide. The turtle walks in the direction it is headed (by default it is 0 degrees, or to the right)
    2. house_turtle.back(distance) makes the turtle walk backwards by distance, a number you provide.
    3. house_turtle.right(angle) turns the turtle right by angle degrees.
    4. house_turtle.left(angle) turns the turtle left by angle degrees.
    5. house_turtle.setheading(angle) takes in a number that represents an angle in degrees (e.g. 0 points to the right, 90 points up, 180 points left, and 270 points south). This command tells the turtle to face in that direction, which you can see by the little arrow representing the turtle.
    6. house_turtle.goto(x, y) takes two numbers representing the screen x and y coordinates and sends the turtle there in a straight line. (0, 0) is the center of the screen.
    7. You can find the complete list of turtle commands here: https://docs.python.org/3.7/library/turtle.html

Comparisons

Operator Description Example
== If the values of two operands are equal, then the condition becomes true. a == b
!= If values of two operands are not equal, then condition becomes true. a != b
> If the value of left operand is greater than the value of right operand, then condition becomes true. a > b
< If the value of left operand is less than the value of right operand, then condition becomes true. a < b
>= If the value of left operand is greater than or equal to the value of right operand, then condition becomes true. a >= b
<= If the value of left operand is less than or equal to the value of right operand, then condition becomes true. a <= b
