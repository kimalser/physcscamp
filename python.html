1 Getting Started

  1. Open Firefox by double-clicking the icon on the desktop, or by clicking on the start menu and typing Firefox into the search bar and clicking on the Firefox application (you can also use another browser, such as Chrome or Safari)
  2. Log in to Firefox Thimble and open the file called code.py. We will be using this file to code in Python!

2 How to Output

  1. Type print("Hello World!"), save the file and click the run button.
    1. print() tells Python what it is supposed to do (in this case, print something out to the screen).
    2. "Hello World!" tells Python what is supposed to be printed to screen. In Python, you need to enclose text phrases in quotes!

3 Variables

  1. Variables are nothing but space in the computer’s memory to store values. You can give it a name and use that name to refer to the value stored in that space. The name can only contain letters, numbers and underscore.
  2. To create a variable, give the variable a name and assign it a value using the = sign.
    1. num = 1 means that we have made a variable called num and given it a value of 1.
    2. Variables can store decimal values too. myfloat = 7.2.
    3. To store text, use single or double quotes. For example, str_hw = "Hello World!" or mystring = 'hello'.
  3. Exercise: Save your name into a variable myname and run print("Hi " + myname)

4 If Statements

  1. In Python you can use if statements in the following way:
    age = 14
    if age > 20:
        print('You are old!')
    Pro Tip! You can use <, > or == to make comparisons. Notice that in an if statement, you have to use double "equals" sign. One "equals" sign tells Python to assign a value to something (x = 5 assigns the value of 5 to the variable x). Two "equals" signs means that you are checking two things to see if they are equal (age == 20 checks to see if the value of the age variable is 20).
  1. If-else statements are an extended form of if statements. The else statement tells Python what should happen if the if-statement is not true.
  2. fruit = 'apple'
    if fruit == 'apple':
        print('Yum! I love apples!')
    else:
        print('No, I want an apple!')
  3. Exercise: Create variables for your name and age. Display "Hi <name>, you can drive!" if your age is more than 16, otherwise display, "Hi <name>, you still have <x> years to go", where x is the difference between your age and 16.
Pro Tip! You are not limited to just checking to see if a variable is equal to a certain value. You can check for many different things. See the Comparison section for more details

5 For Loops

  1. What if you want to print "Hello World!" 5 times? 100 times? For loops are used when you want instructions to run a certain number of times. Let's print "Hello World!" 5 times!
  2. for x in range(5):
        print("Hello World!")
    1. range(5) gives a sequence of numbers starting from 0 and up to (but not including) 5. Each iteration of the loop, it saves the number into variable x. Try replacing "Hello World!" by x and see what happens!
  3. Exercise: Calculate the square of each number from 1 to 10 using a loop.

Comparisons

Operator Description Example
== If the values of two operands are equal, then the condition becomes true. a == b
!= If values of two operands are not equal, then condition becomes true. a != b
> If the value of left operand is greater than the value of right operand, then condition becomes true. a > b
< If the value of left operand is less than the value of right operand, then condition becomes true. a < b
>= If the value of left operand is greater than or equal to the value of right operand, then condition becomes true. a >= b
<= If the value of left operand is less than or equal to the value of right operand, then condition becomes true. a <= b
